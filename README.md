## Features

- サーバーごとの接続権限に基づくアクセス制限 (`bungeecord.server.<name>`)
- `/<server-name>` での接続試行コマンド
- 多言語化可能なサーバー参加/移動/退出/その他各メッセージ
- サーバーから kick された際のサーバー転送機能
- スナップショットサーバーへのダイレクト接続 (BungeeCord の対応状況に依存)
